## 关键决策记录

| 主题 | 决策 |
|------|------|
| 格子用途 | 多张图片排版拼图, 非单图分区 |
| 画布导航 | 右键拖动平移, 画布可超出屏幕 |
| 项目存储 | 文件夹 + 图片复制 + 项目 JSON + 全局索引 |
| 测量工具 | H 型长度 + 限定圆, 预留扩展 |
| 标定方式 | 全局共享; 画线输入或直接输比例 |
| 限定圆交互 | 辅助轨迹线; 平行移动圆心, 垂直移调整半径; 点击确认 |
| 显示缩放 | 全局一致; 首图 contain 初始化; 响应式调整 |
| 图片命名 | 默认原始文件名; 拖到指定格子; 支持重命名 |
| 界面布局 | 顶部工具栏 + 中间画布 + 右侧面板 |
| 项目管理入口 | 启动时选择 + 运行时可切换 |
| 限定圆记录 | 位置 + 直径, 后续补充其他值 |
